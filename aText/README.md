# aText scripts

Download with `brew install atext`. Updating scripts & this readme when I feel like it 

## Ääkköset

- `g'` → `ä`
- `g;` → `ö`

## Arrows

- `;ra` → `→`
- `;la` → `←`
- `;ua` → `↑`
- `;da` → `↓`

## Superscript

- `;sup2` → `²`
- Works for all numbers

## Misc

- `;b` → `•`
- `; shr` → `¯\_(ツ)_/¯`
- `;lenny` → `( ͡° ͜ʖ ͡°)`
- `;tick` → `✓`
